The Problem

Charting shouldn't feel like drowning

Tab switching. Write Mermaid syntax in your editor, paste into a preview tool, tweak, copy back. Repeat endlessly.

Rigid styling. The default output looks like a 2006 intranet diagram. Good luck making it presentable.

No embedding. Want to drop a live chart into your docs or site? You'll need workarounds on top of workarounds.

$ edit chart.md $ open preview-tool.com $ paste → tweak → copy $ paste back → broke again ERROR: life is too short for this
